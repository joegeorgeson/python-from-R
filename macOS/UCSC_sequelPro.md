This is a simple tutorial to access the table browser of [UCSC](https://genome.ucsc.edu/cgi-bin/hgTables) or mirror [UCSC-euro](https://genome-euro.ucsc.edu/cgi-bin/hgTables) via SQL using [SequelPro](https://sequelpro.com/)

Just enter the below connection details;

* Name: UCSC / UCSC-euro
* Host: genome-mysql.soe.ucsc.edu / genome-euro-mysql.soe.ucsc.edu
* Username: genome / genome
* Password: none
